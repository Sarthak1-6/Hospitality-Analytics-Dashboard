# Hospitality Analytics Dashboard

## Project Overview

This project analyzes hospitality industry data to understand hotel performance, booking trends, and revenue insights.
The goal of the project is to help hotel management make data-driven decisions to improve occupancy and revenue.

## Tools Used

* Excel – Data cleaning and preparation
* SQL – Data exploration and analysis
* Power BI – Interactive dashboard creation
* Tableau – Data visualization and reporting

## Key Metrics

* Total Bookings
* Occupancy Rate
* Revenue Generated
* Average Daily Rate (ADR)
* Revenue per Available Room (RevPAR)

## Project Workflow

1. Data cleaning and preprocessing using Excel
2. Data analysis using SQL queries
3. Dashboard creation using Power BI
4. Data visualization using Tableau

## Key Insights

* Weekend bookings are higher compared to weekdays.
* Certain cities generate higher revenue due to higher occupancy.
* Luxury hotels show higher ADR compared to business hotels.
